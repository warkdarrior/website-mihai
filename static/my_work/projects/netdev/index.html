Mihai's Project on
Safety and Security for Network Devices
Embedded devices are often becoming networked, and network devices are becoming more embedded.

Routers

Cisco IOS

IOS is a Cisco proprietary OS developed to run in routers.

Architectures commonly used in Cisco routers: Motorola 68000, MIPS and possibly other RISC processors.

Some known vulnerabilities:

  • 2003-02-21: OSPF Neighbor Buffer Overflow Vulnerability
  • 2003-02-11: ICMP Redirect Routing Table Modification Vulnerability
  • 2002-12-19: EIGRP Announcement ARP Denial Of Service Vulnerability
  • 2002-07-27: TFTP Server Long File Name Buffer Overflow Vulnerability
  • 2002-06-07: Large TCP Scan Denial of Service Vulnerability
For more vulnerabilities, check out Bugtraq's collection of Cisco bugs.
